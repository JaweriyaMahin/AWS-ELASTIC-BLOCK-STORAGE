# AWS EBS Volume

## Overview
This project demonstrates the creation and management of an Amazon EBS Volume in AWS.

## About EBS Volume
An Amazon EBS (Elastic Block Store) Volume is a persistent block storage device that can
be attached to an Amazon EC2 instance. It is used to store operating systems
applications, and data. Unlike instance store, EBS volumes retain data even
when the EC2 instance is stopped.

## Key Features
- Persistent storage
- High durability and availability
- Can be attached to EC2 instances
- Supports resizing
- Can be backed up using EBS Snapshots

## AWS Service Used
- Amazon EC2
- Amazon EBS
